4) Titel: Pong
5) Spielprinzip:

6) Pong ist ein klassisches Spiel welches zu den ältesten Computerspielen überhaupt gehört. Das Prinzip ähnelt Tischtennis, jeder Spieler hat ein „Schläger“ in Form von einem Paddle welches er entlang einer Linie Bewegen kann,  geht der Ball am Schläger vorbei so erhält der Gegner einen Punkt. In meiner Version des Spiels, spielt man ausschließlich gegen einen Spieler und nicht Computer.

7) Programmierkategorien:

1. Interaktive Tastatureingabe: mit „a“ und „d“ kann jeweils der eine Spieler sein Paddle steuern und mit „j“ und „l“ kann der zweite Spieler sein Paddle ändern
2. ASCII Art: Startbildschirm „Pong“
3. Einfache grafische Animation: der Ball und die „Schläger“
4. Grafische animierte und bewegte Objekte: der Ball bewegt sich und prallt vom Paddle ab
5. Multi-Player: man kann gegen einen anderen Spieler spielen

8) Tastaturbelegung: „space“ = Spielstart
				 „a“       = unterer Spieler Links
				 „d“       = unterer Spieler Rechts
				 „j“	    = oberer Spieler Links
				 „l“	    = oberer Spieler Rechts
				 „p“	    = Spiel unterbrechen
				 „q“	    = Spiel beenden
